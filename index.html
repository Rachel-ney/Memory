Memory

Vous avez deux minutes pour retrouver les 14 paires de cartes mélangés sur le plateau.

Cliquez sur une carte pour la retourner, lorsque vous aurez retourné une deuxième carte,
si elles ne sont pas identiques vous aurez 1 seconde pour les mémoriser.
Si la deuxième carte est identique à la première, elles sont alors retiré du jeu.
